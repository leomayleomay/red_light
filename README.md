# RedLight

red_light is a gem used to help you incorporate fancy login in your Rails application effortlessly.

## Installation

Add this line to your application's Gemfile:

    gem 'red_light'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install red_light
